🛠️ Predictive Maintenance Dashboard

📌 Overview

A Streamlit web app that simulates sensor data and detects anomalies using the Isolation Forest algorithm for early failure prediction.

🚀 Features

* Real-time sensor visualization
* Anomaly detection
* Interactive charts (Plotly)
* Alerts for abnormal behavior

🧠 Tech Stack

Python, Streamlit, Pandas, NumPy, Scikit-learn, Plotly

## ▶️ Run the App

```bash
pip install streamlit pandas numpy scikit-learn plotly
streamlit run app.py
```

🎯 Use Case

Helps monitor equipment and detect faults early to reduce downtime.
